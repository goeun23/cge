# ishopcare-merchant-contract

아이샵케어 신규가맹점계약 과제 최고은

[사용자 경험 개선 사항]

1. 데이터 입력 편의성 및 보존

- 세션 기반 데이터 유지: 데이터 저장은 session으로 데이터를 저장
- input field formatting
  - 전체 필드의 데이터 저장시 공백을 제거하여 저장
  - 연락처 하이픈(-) 추가(저장시 제거)
  - 사업자등록번호 길이 제한

2. 안전한 진행 및 에러 방지

- 진입 방어 로직: 필수 정보(대표자명 등)가 없는 상태에서 중간 단계로 로 직접 URL을 통해 접근할 경우, 자동으로 첫 페이지로 리다이렉트
- 데이터 무결성 검사 (Type Guard): 제출 직전에 데이터 구조 검증

3. 실시간 피드백 및 안내

- 즉각적인 유효성 검사: 입력 필드에서 포커스가 벗어나는(Blur) 즉시 유효성을 검사하고, 오류가 있다면 직관적인 filed 하단 에러 메시지를 노출
- 토스트(Toast) 메시지 알림:
  - 기가맹 업체 확인 시 중복된 매장
  - 서버 통신 중 에러 발생 시
- 버튼 상태 시각화: 조건 충족 전 버튼 비활성화 적용
